# Enhanced Skill Cards

This project features enhanced skill cards designed with a professional look, incorporating premium UI elements such as glass effects, hover effects, and animations. The skill cards are responsive and visually appealing, making them suitable for showcasing skills in a portfolio or website.

## Project Structure

```
enhanced-skill-cards
├── src
│   ├── index.html        # HTML structure for the enhanced skill cards
│   ├── styles.css       # Styles for the skill cards, including glass effects and animations
│   └── script.js        # JavaScript for interactive features and animations
├── assets
│   └── icons            # Directory containing icon files used in the skill cards
└── README.md            # Documentation for the project
```

## Features

- **Glass Effect**: The skill cards feature a modern glassmorphism effect that enhances the visual depth.
- **Hover Effects**: Interactive hover effects provide feedback and improve user engagement.
- **Animations**: Smooth animations are applied to elements for a polished user experience.
- **Responsive Design**: The cards are designed to be responsive, ensuring they look great on all devices.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/MUREEDSAJJAD512/mureedsajjad512.github.io.git
   ```

2. Navigate to the project directory:
   ```
   cd enhanced-skill-cards
   ```

3. Open the `src/index.html` file in your web browser to view the enhanced skill cards.

## Usage

- Customize the skill cards by editing the `src/index.html` file to add or modify skills.
- Update the `src/styles.css` file to change styles or effects as desired.
- Use the `src/script.js` file to add any additional interactive features or animations.
